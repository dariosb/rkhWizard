Detailed Description

Defines the data structure used to maintain information that allows the timer-handling facility to update and expire software timers.

RKH prohibits an application from explicitly modifying the RKHT_T structure. The RKH's timer structures can be located anywhere in memory, but it is most common to make it a global structure by defining it outside the scope of any function. An RKH timer is created when an application timer is declared with the RKHT_T data type. The following listing declares "my_timer" timer:

    RKHT_T my_timer;

RKH_TNT_T RKHT_T::period

Number of ticks for all timer expirations after the first (expiration period). A zero for this parameter makes the timer a one-shot timer, otherwise, for periodic timers, any value in range.
