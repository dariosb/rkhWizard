Detailed Description

Constant parameters of state machine.

The constant key parameters of a state machine are allocated within. ROMRKH_T is a ROM base structure of RKH_T.

rkhui8_t ROMRKH_T::id

ID of state machine application.

ID of state machine application. This number allows to uniquely identify a state machine. When a particular application requires runtime tracing (native tracing features), the option RKH_SMA_EN_ID must be set to one.
